# asteroids

asteroids is my second [Boot.dev](https://www.boot.dev) project. This project uses object oriented code python code to replicate a simple version of Asteroids.

## Setup

```bash
# Install the specified requirements
pip install -r requirements.txt

# Create a virtual environment named 'venv'
python3 -m venv venv

# Activate the virtual environment
source venv/bin/activate
```


## Usage
```bash
python3 main.py
```
